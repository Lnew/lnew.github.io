TensorFlow初探

本文共计628个字 | 您是第 位看到它们的小伙伴

安装虚拟环境 virtualenv

pip install virtualenv

创建虚拟环境

virtualenv –system-site-packages -p python .\Lzz\TensorFlow

激活虚拟环境

cd .\Lzz\TensorFlow\Scripts
activate

安装 tensorflow

pip install tensorflow

这里安装好使用的时候如果提示ImportError: DLL load failed: 找不到指定的模块。

pip uninstall tensorflow 把原来最新版本卸载 pip install tensorflow==1.9
pip uninstall protobuf 卸载 pip install protobuf==3.6.0

测试了一会，发现只要下面的是3.6.0就可以，不要管tensorflow 1.12.0 has requirement protobuf>=3.6.1, but you’ll have protobuf 3.6.0 which is incompatible.这个报错

pip install protobuf==3.6.0
这里搞了大半天，太他妈坑了！！！

退出虚拟环境

deactivate
